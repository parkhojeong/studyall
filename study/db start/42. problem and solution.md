# problem1: product discount price calculation
from products, price discounted 15%, alias sale_price
select name, price, sale_price
```sql
select 
    name, price, (price * 0.85) as sale_price
from 
    products;
```

# problem2: customer info merge good to show
from customers, concat name and address with ' - ' using CONCAT_WS
alias customer_info
```sql
select 
    CONCAT_WS(' - ', name, address) as customer_info
from 
    customers;
```

# problem5: split and analyze email address
from customers, extract id from email, calculate length of id
id is string before @
use SUBSTRING_INDEX(), CHAR_LENGTH(), alias user_id, id_length
```
email                    user_id       id_length
sejong@exmaple.com       sejong        6
yisunsin@example.com     yisunsin      8
youngsil@example.com     yongsil       8
```

```sql
select 
    email, 
    SUBSTRING_INDEX(email, '@', 1) as user_id, 
    CHAR_LENGTH(SUBSTRING_INDEX(email, '@', 1)) as id_length
from customers;
```
