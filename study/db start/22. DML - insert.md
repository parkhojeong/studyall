* DML - insert

* INSERT GRAMMAR
```sql
INSERT INTO table_name (column1, column2, column3, ...)
VALUES (value1, value2, value3, ...);
```

1. insert all columns
```sql
INSERT INTO customers 
values(NULL, 'name', 'b@am.com', 'q23', '123', '2025-11-11 10:10:10', NULL);
```
* PK NULL is assigned incremented

2. insert part of columns
```sql
INSERT INTO customers (name, email, password, address)
values('name3', 'a@a.com', '1234', '1234');

INSERT INTO products(name, price, stock_quantity)
values('product3', 20000, 100);
```
* PK NULL is assigned NULL
* created_at is assigned current time(default)
* updated_at is assigned current time(default)

* practice: insert part of columns
- insert all columns -> not compatible table column changes

INSERT INTO products(name, price, stock_quantity) values
