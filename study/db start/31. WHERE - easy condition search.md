* WHERE - easy condition search
- BETWEEN(N,M): N <= x <= M
  - NOT BETWEEN(N,M): (x < N) OR (M < x)
- IN(N,M,...): (x = N) OR (x = M) OR ...
  - NOT IN(N,M,...): (x != N) AND (x != M) AND ...
- LIKE: partial characters pattern matching
  - % : [any character]* - 0 or more any character
    - 'kim%': start with 'kim'
    - '%kim': end with 'kim'
    - '%kim%': contains 'kim'
  - _ : any 1 character
    - 'k_m': kXm (X is any character)
- IS NULL
