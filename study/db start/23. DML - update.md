* DML - update

* UPDATE GRAMMAR
```sql
UPDATE table_name
SET column1 = value1, column2 = value2, ...
WHERE condition;
```

* SQL safe update mode
- SELECT @@SQL_SAFE_UPDATES;
- SET @@SQL_SAFE_UPDATES=0; // 1 = ON, 0 = OFF
- force UPDATE or DELETE with WHERE clause using pk, or LIMIT clause

summary
- MySQL server(global): default off(0)
- application(connection): default off(0)
- MySQL workbench: default on(1)
  - Settings > SQL Editor > SQL Safe Updates

practice caution
- select the same where clause before update or delete
