* summary
- SLECT
  - get data from database
  - * : all columns
  - column1, column2,
  - AS for alias
- WHERE - basic search
  - filter matching condition
  - =, !=, >, <
  - AND, OR, NOT
- WHERE - easy condition search
  - BETWEEN: range search
  - IN: list include?
  - LIKE: pattern matching
    - % : [ch]*
    - _ : [ch]
  - IS NULL: comparison with null
    - caution: = operator not work with null
- ORDER BY - sort
  - result set order 
  - ASC(default), DESC
  - column1(, column2)*
- LIMIT - count limitation
  - result set size
  - meaningful with ORDER BY
  - LIMIT offset, count: for pagination implementation
  - offset = (page - 1) * count per page
- DISTINCT - remove duplicated rows
  - select result set without duplicated rows
  - equality comparison
  - multiple columns
- NULL - unknown value
  - unknown value. different from 0 or space
  - use IS NULL, IS NOT NULL for NULL comparison
  - MySQL deal with NULL as smallest value
  - NULL ordering: ORDER BY (column IS NULL) DESC
