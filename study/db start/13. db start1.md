# CREATE TABLE table_name(): table creation
```sql
-- syntax
CREATE TABLE tablename(
    columnname datatype [PRIMARY KEY]
    -- ...
);
```
create sample table
```sql
CREATE TABLE sample(
    product_id INT PRIMARY KEY,
    name VARCHAR(100),
    price INT,
    stock_quantity INT,
    release_date DATE
);
```
    
data type
- INT: integer
- VARCHAR: variable length character
- DATE: date
