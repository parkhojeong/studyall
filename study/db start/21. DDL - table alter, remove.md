* DDL - table alter, remove
- ALTER TABLE table_name ADD column_name data_type;
- ALTER TABLE table_name MODIFY column_name data_type; 
- ALTER TABLE table_name DROP COLUMN column_name;

- DESC table_name: show table structure

* practice caution
- ALTER TABLE consume many resources
- recent MySQL version, adding new column not matter
- check lock and issue in documentation for database version.

* DROP TABLE vs TRUNCATE TABLE
- DROP TABLE: delete data and table structure
- TRUNCATE TABLE: delete data only
  - characteristic
    - reset table (delete apply single row and log, can rollback) 
    - reset AUTO_INCREMENT value. (delete not reset)
- usage 
  - test data reset: TRUNCATE
  - delete table: DROP 

* ignore foreign key constraint
- SET FOREIGN_KEY_CHECKS=0; -> ignore
- SET FOREIGN_KEY_CHECKS=1; -> restore
- 'SET XX' valid in session
