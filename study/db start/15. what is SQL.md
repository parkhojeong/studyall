* sql command 4 types
1. DDL(Data Definition Language)
   - purpose: manage data structure
   - command
     - CREATE: DATABASE, TABLE
     - ALTER: TABLE
     - DROP: DATABASE, TABLE
2. DML(Data Manipulation Language)
   - purpose: manipulate data in table
   - command
     - INSERT
     - SELECT
     - UPDATE
     - DELETE
3. DCL(Data Control Language)
   - purpose: grant or revoke access permission. for security
   - command
     - GRANT
     - REVOKE 
4. TCL(Transaction Control Language)
   - purpose: bundle multiple sql(DML) commands into transaction
   - command
     - BEGIN: start transaction
     - COMMIT: commit transaction. all changes will be saved.
     - ROLLBACK: rollback transaction. all changes will be reverted.
