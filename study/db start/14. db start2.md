* CRUD
# INSERT INTO table_name(column1, .., columnN) values(value1, .., valueN) 
```sql
INSERT INTO sample (product_id, name, price, stock_quantity, release_date)
VALUES (1, 'Product A', 100, 50, '2023-01-01');
```

# SELECT * FROM table_name
```sql
SELECT * FROM sample;
```

# UPDATE table_name SET column1=value1 WHERE condition
```sql
UPDATE sample SET price = 100 WHERE product_id = 1;
```

# DELETE FROM table_name WHERE condition
```sql
DELETE FROM sample WHERE product_id = 1;
```

* naming convention
- keyword: upper case
- non-keyword: lower case + underscore
