* database terms organize
- database: structured data collection for specific purpose
- DBMS: software that manage database(creation, definition, sharing, usage, management)

* file system vs DBMS
- DBMS: save, query, edit, delete data
- data store: save specific file format for dbms system 
- abstraction: provide SQL interface 
- prevent direct access to data store: direct access damage data. should access from DBMS
-> DBMS provide abstraction for data management

* DBMS
1. data definition language: DDL
  - data type, relation, constraint
2. data manipulation language: DML
  - insert, select, update, delete
  - easy use by SQL(structured query language)
3. security, concurrency control, transaction management
  - security: permission control(access, modify)
  - concurrency control: provide order control, locking mechanism
  - transaction management: guarantee atomic operation (All or nothing)
4. duplication minimization and consistency maintenance
  - normalization 
5. backup and recovery
  - system failure(hard list broken, blackout), damage or loss by user
  - log file backup and recovery
