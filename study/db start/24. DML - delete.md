DELETE vs TRUNCATE
- DML vs DDL
- line vs table
- slow vs fast
- AUTO_INCREMENT not reset vs reset
- rollback: transaction rollback possible vs not

when and what to use 
- conditional delete: delete
- clear table(ex. testing data reset): truncate
