* conceptual modeling practice - word dictionary writing

```
+-----------+-----------+----------------+------------+------------------------------------------------------------+-----------------------------+
| 분류       | 명칭       | 전체 영문명       | 축약어     | 설명                                                          | 관련 시스템 요소            |
+-----------+-----------+----------------+------------+------------------------------------------------------------+-----------------------------+
| 엔티티      | 회원       | member         | -          | 서비스를 이용하는 고객. member로 용어 통일.                        | member 테이블               |
|           | 상품       | product        | -          | 판매하는 물건 또는 서비스.                                       | product 테이블              |
|           | 주문       | order          | -          | 회원의 상품 구매 요청 행위. SQL 예약어로 orders 사용.               | orders 테이블               |
|           | 주문 항목   | order_item     | -          | 주문에 포함된 개별 상품 정보. 주문-상품 M:N 관계 해소.                | order_item 테이블           |
|           | 배송       | delivery       | -          | 주문된 상품의 물리적 이동 정보.                                   | delivery 테이블             |
|           | 결제       | payment        | pay        | 주문에 대한 지불 정보.                                          | payment 테이블              |
+-----------+-----------+----------------+------------+------------------------------------------------------------+-----------------------------+
| 주요 속성 | 식별자        | identifier     | id         | 데이터를 고유하게 식별하는 번호. [엔티티명]_id 형식 사용.              | member_id, product_id       |
|           | 이름, 명    | name           | -          | 사람, 상품 등 대상을 지칭하는 명칭.                                | member_name, product_name   |
|           | 가격       | price          | -          | 현재 판매가 또는 주문 시점 가격. 문맥에 따라 구분함.                   | order_price                 |
|           | 금액       | amount         | -          | 수량과 단가가 곱해진 총액. 예) 총 주문 금액                          | total_amount                |
|           | 재고       | stock          | -          | 상품의 보유 물량.                                              | stock_quantity              |
|           | 수량       | quantity       | -          | 개수나 양. 재고 수량, 주문 수량 등.                                | stock_qty, order_qty        |
|           | 개수, 횟수  | count          | -          | 개별 항목을 세는 행위, 이벤트 발생 횟수, 데이터 총 개수 등.             | visit_count, click_count    |
|           | 상태       | status         | -          | 주문, 배송 등 엔티티의 현재 상태를 나타내는 값.                       | order_status, delivery_stat |
|           | 주소       | address        | addr       | 위치 정보. 기본 주소(addr), 배송지(ship_addr) 등.                 | member.addr, ship_addr      |
|           | 비밀번호    | password       | -          | 로그인 시 사용하는 암호화된 비밀번호.                                | member.password             |
|           | 배송(접두)  | shipping       | ship       | 배송과 관련된 속성을 나타낼 때 접두사로 사용.                         | ship_addr, ship_fee         |
|           | 로그인      | login          | -          | 시스템에 접속하는 행위 및 관련 정보.                               | member.login_id             |
|           | 수단       | method         | -          | 행위의 방식. 예) 결제 수단                                       | pay_method                  |
|           | 번호       | number         | no         | 식별 목적 외의 일반 번호. 예) 운송장 번호                           | tracking_no                 |
|           | 운송장번호   | tracking_no    | -          | 배송 추적을 위한 고유 번호.                                      | tracking_no                 |
+-----------+-----------+----------------+------------+------------------------------------------------------------+-----------------------------+
| 행위/      | 생성      | create          | -          | 데이터가 최초로 생성된 시점.                                      | created_at                  |
| 수식어      | 수정      | update          | -          | 데이터가 마지막으로 변경된 시점.                                   | updated_at                  |
+-----------+-----------+----------------+------------+------------------------------------------------------------+-----------------------------+
```
