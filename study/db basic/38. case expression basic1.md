* case expression basic1

* simple case expression
```
CASE column_or_expression
  WHEN value1 THEN result1
  WHEN value2 THEN result2
END
```

```sql
select order_id,
       CASE status
           WHEN 'PENDING' THEN 'ORDER PENDING'
           WHEN 'COMPLETED' THEN 'PAY COMPLETED'
           WHEN 'SHIPPED' THEN 'DELIVERED'
           WHEN 'CANCELLED' THEN 'ORDER CANCELLED'
           ELSE 'UNKNOWN'
       END as status_desc
from orders;
```

* searched case expression
- evaluation is done from top to bottom
```
CASE 
    WHEN condition1 THEN result1 
    WHEN condition2 THEN result2
    ... 
    ELSE resultN 
END
```
```sql
use my_shop2;
select name,
       price,
       CASE
           WHEN price >= 100000 THEN 'HIGH PRICE'
           WHEN price >= 30000 THEN 'MEDIUM PRICE'
           ELSE 'LOW PRICE'
           END as price_label
from products;
```
