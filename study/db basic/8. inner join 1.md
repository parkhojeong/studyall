* inner join 1
- requirement
  - from orders where status = 'COMPLETE'
  - select user(user_id, name), order_date
```sql
select 
    users.user_id, 
    users.name, 
    orders.order_date 
from orders 
inner join users 
    on orders.user_id = users.user_id
where orders.status = 'COMPLETE';
```

* inner join concept
- combine child.parent_id = parent.parent_id
- INNER can be omitted. (INNER) JOIN
- child table + parent table on child.parent_id = parent.parent_id
  - attach parent table to child table 

```sql
select 
    user_id  
from orders 
inner join users 
    on orders.user_id = users.user_id;
-- Error Code: 1052. Column 'user_id' in field list is ambiguous
```

* join logical execution order
1. FROM / JOIN : virtual table creation
2. WHERE -> filter rows
3. SELECT
