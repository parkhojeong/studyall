* cross join
- requirement: all combination of two tables

* CROSS JOIN concept and Cartesian product
- row x row
```sql
select
    concat(c.color,'-',s.size) as product_name,
    s.size,
    c.color
from sizes s
cross join colors c;
```

* INSERT INTO ... SELECT
```sql
CREATE TABLE product_options(
    option_id BIGINT AUTO_INCREMENT,
    product_name VARCHAR(255) NOT NULL,
    size VARCHAR(10) NOT NULL,
    color VARCHAR(20) NOT NULL,
    PRIMARY KEY(option_id)
);
```

```sql
INSERT INTO product_options(product_name, size, color)
SELECT 
    concat(c.color,'-',s.size) as product_name,
    s.size,
    c.color
FROM sizes s
CROSS JOIN colors c;
```
