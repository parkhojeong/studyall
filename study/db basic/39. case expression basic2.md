* case expression basic2

* searched case expression caution: when clause order
- evaluation order is from top to bottom

* case expression usage
- select clause, order by clause, group by clause, where clause
- ex. order by case when ...
```sql
select name,
       price,
       CASE 
           WHEN price >= 100000 THEN 'HIGH PRICE'
           WHEN price >= 30000 THEN 'MEDIUM PRICE'
           ELSE 'LOW PRICE'
       END as price_label
from products
order by 
	CASE 
		WHEN price >= 100000 THEN 1
 	    WHEN price >= 30000 THEN 2
	    ELSE 3
	END ASC,
    price desc
```
