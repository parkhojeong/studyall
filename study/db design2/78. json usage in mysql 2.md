* json data update

- json_set(if exists, update, else add)
```sql
update product_json
set attributes = json_set(attributes, 
                          '$.wireless_charging', true,
                          '$.storage', 1024)
where id = 1;

select name, attributes from product_json where id = 1;
```

- json_insert(if not exists, add)
```sql
update product_json
set attributes = json_insert(attributes, 
                          '$.color', 'new color3',
                          '$.fast_charging', true)
where id = 1;

select name, attributes from product_json where id = 1;
```

- json_replace(if exists, replace, else nothing)
```sql
update product_json
set attributes = json_replace(attributes, 
                          '$.storage', 512,
                          '$.new_feature', true)
where id = 1;

select name, 
       attributes ->> '$.storage' as storage,
       attributes ->> '$.new_feature' as new_feature
from product_json where id = 1;
```

summary

| function     | description                |
|--------------|----------------------------|
| json_set     | if exist, update, else add |
| json_insert  | if exist, nothing          |
| json_replace | if not exist, nothing      |

* summary
json grammar core
- {}, []
- 6 types: string, number, boolean, null, object, array
- key must be wrapped with ""
mysql json core functions
- json_object, json_array: json creation
- ->, ->> : extract value from json
  - ->: json type
  - ->> : string type
- json_set, json_insert, json_replace: update json
- json_remove: delete key from json
- json_contains: check if key exists in json
