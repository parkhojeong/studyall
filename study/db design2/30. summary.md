| column name   | type         | description                                 |
|---------------|--------------|---------------------------------------------|
| change_type   | VARCHAT(50)  | chagne type(PRICE_CHANGE,STATUS_CHANGE,etc) |

* audit column
for detail audit tracking

| column name   | type        | description        |
|---------------|-------------|--------------------|
| source_system | VARCHAR(50) | what system change |
| client_ip     | VARCHAR(50) | client ip          |
usage situation: when many systems change. when security audit is required.

- all tracking columns added example
```sql
CREATE TABLE product (
  -- 비즈니스 컬럼
  product_id BIGINT PRIMARY KEY AUTO_INCREMENT,
  name VARCHAR(200) NOT NULL,
  price INT NOT NULL,
  stock_quantity INT NOT NULL DEFAULT 0,
  status VARCHAR(20) NOT NULL DEFAULT 'ACTIVE',
 
  -- 기본 변경 추적 컬럼
  created_at DATETIME NOT NULL DEFAULT CURRENT_TIMESTAMP,
  created_by VARCHAR(100) NOT NULL,
  updated_at DATETIME NOT NULL DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
  updated_by VARCHAR(100) NOT NULL,
 
  -- 변경 사유 컬럼
  change_type VARCHAR(50),
  change_reason VARCHAR(500),
 
  -- 감사(Audit) 컬럼
  source_system VARCHAR(50),
  client_ip VARCHAR(50)
);
```

* content summary
why data update history design is required
- data update history is a necessary factor
- in production, variable question is always asked
- history is required for error tracking, audit prepares, customer claim
original table without history - problem analysis
- in the original table, when the update is done, the previous value is lost
change tracking columns - basic
- created_at, created_by, updated_at, updated_by
- necessary column to basic table design
change tracking columns - change reason
- change_type, change_reason
- change_type: standardize code for system usage, search efficiency
- change_reason: write detail reason for employee to understand
change tracking columns - audit
- source_system, client_ip
- source_system: a problem caused is specified quickly by which a system makes a change
