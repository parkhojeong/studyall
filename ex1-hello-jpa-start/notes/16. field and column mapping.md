* mapping annotations
- @Column: column mapping
- @Temporal: date type mapping
- @Enumerated: enum type mapping
- @Lob: BLOB, CLOB mapping
- @Transient: ignore column mapping

* @Column properties
- name: column name
- insertable, updatable: default true
- only for DDL
  - nullable: default true
  - unique: same as uniqueConstraints in @Table
    - constraint name cannot be customized
  - columnDefinition
  - length: only for string type
    - default: 255
  - precision, scale: for BigDecimal, BigInteger

* @Enumerated
- java enum type mapping
- value
  - EnumType.ORDINAL(default)
  - EnumType.STRING(recommended)

* Temporal
- for java.util.Data, java.util.Calendar mapping
- it can be omitted with LocalData, LocalDateTime(hibernate supported)
- value
  - TemporalType.DATE: db date type(YYYY-mm-DD)
  - TemporalType.TIME: db time type(HH:MM:SS)
  - TemporalType.TIMESTAMP: db timestamp type(YYYY-mm-DD HH:MM:SS)

* @Lob
- db BLOB, CLOB type mapping
- no property
- by field type
  - String, char[], java.sql.CLOB: CLOB mapping
  - byte[], java.sql.BLOB: BLOB mapping

* @Transient
- ignore column mapping
- field only used in memory
