* Jpa data type classification
- Entity type
  - class with @Entity
  - it can be tracked
- Value type
  - int, Integer, String
  - it cannot be tracked

* value type classification
- primitive type
  - java primitive type(int, double)
  - wrapper class(Integer, Long)
  - String
- embedded type(composite value type)
- collection value type

* primitive type
- it's lifecycle depends on the entity
- it must not be shared
- java primitive type is immutable(copy and assign)
  - wrapper class is also immutable
