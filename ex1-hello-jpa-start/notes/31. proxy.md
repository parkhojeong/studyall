* proxy basic
- em.find(): find from db
- em.getReference(): get proxy entity

* proxy characters
- proxy instance has a target reference to the real instance
- if a proxy method is called, the target instance method is called

* proxy instance initialization
- Member member = em.getReference(Member.class, "id1")
- member.getName()
1. getName() is called
2. proxy instance request initialization to persistence context
3. persistence context load member from db
4. a real entity is created
5. target.getName() is called

* proxy characters
- a proxy instance is initialized when it is used at once
- proxy instance extends the real instance
  - use `instance of` to check the type instead of ==
- first if proxy is used, after find, getReference use proxy
- first if target is used, after find, getReference use target
