* entity mapping overview
- class and table mapping: @Entity, @Table
- field and column mapping: @Column
- pk mapping: @Id
- relationship mapping: @ManyToOne, @JoinColumn

* @Entity
- class with @Entity is managed by JPA, it is called entity
- caution
  - a default constructor is required(public or protected and no parameter)
  - final, enum, interface, inner class are not allowed
  - the final field is not managed by JPA
- attribute(name)
  - entity name in jpa
  - default(recommended): class name in lower case

* @Table 
attributes
  - name: table name mapped to entity
    - default(recommended): entity name
  - catalog
  - schema
  - uniqueConstraints
