* @MappedSuperclass
- when common mapping info is needed
  - provides mapping info to child classes
  - ex. created_at, created_by, updated_at, updated_by
- not 
  - inheritance relationship mapping
  - entity, table mapping
- recommend abstract class
- notes: class with @Entity can extend claas with @Entity or @MappedSuperclass only

```java
import jakarta.persistence.Entity;

@MappedSuperclass
class BaseEntity {
    @Id
    @GeneratedValue
    private Long id;

    private String name;
}

@Entity
class Member extends BaseEntity {
}
```
